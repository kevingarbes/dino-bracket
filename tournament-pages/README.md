# Tournament Pages Workspace

Use these folders to organize redesign work by tournament. The folder paths are only for local project organization; the public URLs listed below should remain unchanged at relaunch.

| Folder | WordPress intro URL | Standalone bracket URL |
| --- | --- | --- |
| `dinosaurs/` | `/dinosaurs/` | `/dinosaur-tournament.html` |
| `box-office-hits/` | `/box-office-hits/` | `/box-office-hits-tournament.html` |
| `cryptids/` | `/cryptids/` | `/cryptid-tournament.html` |
| `dog-breeds/` | `/dog-breeds/` | `/dog-breed-tournament.html` |
| `flowers/` | `/flowers/` | `/flower-tournament.html` |
| `popes/` | `/popes/` | `/pope-tournament.html` |
| `sharks/` | `/sharks/` | `/shark-tournament.html` |

Suggested files as each tournament is redesigned:

- `intro.html` for the WordPress intro page design.
- `bracket.html` for the standalone tournament app design.
- `articles/{division}/` for article redesigns grouped by the tournament's four live divisions.
- `notes.md` for content, assets, and URL-specific requirements.

See `articles-map.md` for the full division folder map.
